# 🌉 BridgeAI
### The Agent That Bridges Every Other Agent

> *AI is everywhere. But almost nobody knows how to actually use it.*

BridgeAI is a proactive AI literacy agent — delivered as a home screen widget — that lives inside the technology behavior people already have, meets them where they are, and teaches them to harness AI by doing, not by lecturing.

**Built for the 50% of the workforce that has access to AI but doesn't know how to use it.**

## The Solution

BridgeAI is a **proactive AI guidance agent** that:

- Lives on your **home screen widget** — zero friction, always present
- Watches what you're trying to accomplish and surfaces the exact AI tool or prompt that solves it
- **Teaches by doing** — you learn AI in context, not in a classroom
- Adapts to your skill level automatically — beginner to advanced, seamlessly
- Acts as your **personal AI co-pilot** across every app, task, and goal

No surveillance. No private data. No legal complexity. Just intelligence meeting people where they already are.

---

## The Three Agent Layers

### 🔍 Context Agent *(always on)*
Understands what you're working on right now — a document, a deadline, a question, a problem — and quietly identifies where AI could help. No input required.

### 🧭 Guidance Agent *(proactive)*
Surfaces the right tool, prompt, or workflow for your exact situation. Not a generic suggestion — a specific, actionable recommendation you can use in seconds.

### 📈 Growth Agent *(adaptive)*
Tracks your AI skill progression over time. As you get more capable, the recommendations evolve. You never get suggestions below your level or above your reach.

The home screen is the most valuable real estate on any phone. BridgeAI owns it — not with static information, but with **live intelligence**.

## Tech Stack

**Mobile**
- React Native + Expo
- iOS Widget Extension + Android App Widget
- Expo TaskManager for background context awareness
- AsyncStorage for local skill progression data

**Agent Layer**
- Node.js agent runtime
- Anthropic Claude API — context understanding, guidance generation, adaptive coaching
- Multi-agent orchestration: Context → Guidance → Growth pipeline
- On-device processing first — minimal data leaves the device

**Backend** *(minimal by design)*
- Supabase — anonymized skill progression metadata only
- No content data stored server-side
- No behavioral logs

**AI/ML**
- Claude Sonnet — real-time context understanding and guidance
- Skill progression model — adaptive difficulty and recommendation engine
- Tool knowledge graph — maps tasks to AI tools and prompts

## The Ethical Commitments

1. **No content surveillance** — BridgeAI never reads your documents, messages, or files
2. **No behavioral profiling** — skill data stays on-device unless you opt into sync
3. **Free core features always** — the basic guidance layer is never paywalled
4. **Human first** — BridgeAI amplifies human capability, never replaces human judgment
5. **Transparent recommendations** — always explains *why* it's suggesting something
